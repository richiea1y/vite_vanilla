最佳旅遊路線規劃器

題目描述：

實現一個函數 findBestTravelRoute，該函數能夠根據給定的旅遊景點數據，找出從起點到終點的最佳路線。函數需要考慮交通時間、平均消費和景點評分，並根據成本權重和評分權重來平衡這些因素。

函數簽名：

function findBestTravelRoute(spots, start, end, costWeight = 0.7, ratingWeight = 0.3)

注意：

  • 使用 Decimal.js 來處理計算，以避免浮點數精度問題。
  • 請先安裝 npm install decimal.js
  • 權重參數（costWeight 和 ratingWeight）代表了旅行者的心理預期，對結果有重要影響。
